# Gaurav Visual Portfolio

A premium, animated personal portfolio website for UX, product, design systems, and AI transformation leadership.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy

This project can be deployed to Vercel, Netlify, GitHub Pages, or any static hosting platform that supports Vite.

## Personalize

Update the following in `src/main.jsx`:

- Email in the contact button
- Case study titles and descriptions
- External links
- Images
